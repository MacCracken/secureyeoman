Enterprise Security

RBAC (Admin/Operator/Auditor/Viewer), JWT + API key auth, mTLS, AES-256-GCM encryption at rest, and five sandboxing modes (Landlock, seccomp, namespaces, gVisor, WASM). Unified SecretsManager — env, keyring, file, Vault/OpenBao, or auto. TLS lifecycle with auto-generated certs and expiry monitoring. Organizational Intent enforces hard boundaries, soft policies, and authorized actions at the pipeline level — with optional OPA sidecar integration.

  • RBAC + SSO/OIDC (Okta, Azure AD, Auth0)
  • Vault / OpenBao secrets backend
  • Five sandboxing modes
  • 18-pattern credential redaction on every response
  • Organizational Intent — hard boundaries & OPA policies

Audit & Compliance

HMAC-SHA256 cryptographic audit chain — every tool call, model response, and system event is logged, verifiably chained, and exportable. Prometheus metrics with Grafana dashboards for real-time production monitoring.

  • Cryptographic integrity (HMAC-SHA256)
  • Prometheus + Grafana dashboards
  • JSON / HTML / CSV export
  • Immutable, tamper-evident logs

Security Operations

37 MCP tools for active network evaluation: device discovery, port scanning, SSH automation, NetBox IPAM, NVD/CVE lookups, and PCAP capture. Twingate zero-trust remote access — reach private MCP servers through an encrypted tunnel with no firewall ports opened. Kali Security Toolkit included.

  • 37 network evaluation tools
  • Twingate zero-trust tunnel (13 tools)
  • NVD / CVE vulnerability lookups
  • Kali toolkit (nmap, nuclei, sqlmap)

120+ MCP Tools & Integrations

31 platform integrations: Telegram, Discord, Slack, WhatsApp, Signal, MS Teams, GitHub, GitLab, Gmail, Google Calendar, Jira, Notion, AWS, Azure DevOps, and more. 120+ built-in MCP tools, 9 resources, and 4 prompts — with SSRF protection, encrypted credentials, and Markdown-for-Agents content negotiation (Accept: text/markdown, Content-Signal enforcement).

  • 31 platform integrations
  • 120+ tools · 9 resources · 4 prompts
  • SSRF protection + encrypted creds
  • Markdown-for-Agents content negotiation

Multi-Agent & Workflows

A2A protocol with E2E encryption and mDNS peer discovery. Sub-agent delegation with budget and depth controls. Agent swarms (sequential, parallel, dynamic). DAG workflow engine with 9 step types, Mustache data-flow templates, and a ReactFlow visual builder — no other framework in this category ships this.

  • A2A protocol (E2E encrypted)
  • Agent swarms + sub-agents
  • DAG workflows — 9 step types
  • ReactFlow visual builder

Cognitive Memory

Vector semantic search (FAISS/Qdrant/ChromaDB) fused with tsvector full-text search via Reciprocal Rank Fusion — exact-term recall meets semantic search. 800-token overlapping document chunks. Proactive context compaction at 80% window fill. Self-repairing task loop with diagnostic recovery prompts.

  • Hybrid FTS + vector (RRF)
  • Content-chunked document indexing
  • Proactive context compaction
  • Self-repairing task loop

Developer Experience

Sandboxed code execution (Python, Node.js, shell). Desktop control — screen capture, keyboard/mouse, clipboard — gated by explicit consent and audit trail.

  • 24 CLI commands + full-screen TUI

Flexible Deployment

Single ~80 MB binary (Linux x64/arm64, macOS arm64), Docker, or Kubernetes with Helm, HPA, PDB, and NetworkPolicies. Three network modes — Local, LAN, or Public TLS — with a live badge in the dashboard. Multi-user workspaces, voice I/O (10 TTS + 7 STT providers), and portable skill marketplace sync.

Get Started in Minutes

Deploy your personal AI Yeoman with just a few commands.

terminal 
# Install SecureYeoman (Linux/macOS)
$ curl -fsSL https://secureyeoman.ai/install | bash

# First-time setup
$ secureyeoman init

# Start the server
$ secureyeoman start

# Access the dashboard
$ open http://localhost:18789
terminal 
# Clone the repository
$ git clone https://github.com/MacCracken/secureyeoman.git
$ cd secureyeoman

# Install dependencies
$ npm install

# Configure environment
$ cp .env.example .env
# Edit .env with your API keys

# Start SecureYeoman
$ npm run dev
terminal 
# Start with Docker Compose
$ docker compose up -d

# With MCP service
$ docker compose --profile mcp up -d

# Fresh start
$ docker compose down -v
$ docker compose up -d
terminal 
# Deploy with Helm (EKS, GKE, AKS)
$ helm install secureyeoman deploy/helm/secureyeoman \
    --namespace secureyeoman --create-namespace \
    --set secrets.postgresPassword=your-password \
    --set database.host=your-db.example.com

# Production deployment
$ helm install secureyeoman deploy/helm/secureyeoman \
    -f deploy/helm/secureyeoman/values-production.yaml \
    --namespace secureyeoman-production --create-namespace

# Verify
$ helm test secureyeoman --namespace secureyeoman
terminal

- Node.js 20 LTS+
